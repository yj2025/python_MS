import sys
sys.executable		현재 실행 중인 파이썬 인터프리터의 경로를 가져온다.

import subprocess		subprocess 모듈은 외부 프로그램을 실행할 수 있게 해준다.
			이 코드에서는 subprocess.check_call()을 사용해 외부 명령어를 실행하고 있다.

subprocess.check_call([sys.executable, "-m", "pip", "install", "konlpy"])	subprocess.check_call()은 인자로 전달된 명령어를 실행하는 함수다. 명령어가 정상적으로 실행되면 반환값 없이 종료되며, 만약 오류가 나면 CalledProcessError를 발생시킨다.

sys.executable		현재 실행 중인 파이썬 인터프리터의 경로를 가져온다.
"-m"			파이썬 모듈을 실행할 때 사용하는 옵션이다. 	이 옵션은 지정된 모듈을 스크립트처럼 실행할 수 있게 해준다.
"pip"			파이썬 패키지 관리자인 pip를 실행하는 것이다.
"install"			pip에게 패키지를 설치하라는 명령어다.
"konlpy"		 	설치할 패키지 이름이다. 이 경우에는 konlpy라는 패키지를 설치하려는 것이다.
